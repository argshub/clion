# Origin of C

C is a powerful general-purpose programming language. It is fast, portable and available in all platforms. If you are new to programming, C is a good choice to learn programming. Because all other high levelprogramming evolve fron C language. C was first invented and impleneted by Dennis Ritchie. It's the only ISO certified language.


#Level of C

C can manipulate computer low level bit, byte and address. Those can be done by low level Assemby Language. Also it's have high level langauge featur like function, union, structure. So C treated as a mid level language.
